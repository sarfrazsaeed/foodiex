# 🍕 FoodieX — Pakistan's Food Delivery App

> Advanced food delivery frontend — React 18 + TypeScript + Vite + GSAP + Framer Motion

[![Live Demo](https://img.shields.io/badge/Live-Demo-E8192C?style=for-the-badge)](https://sarfrazsaeed.github.io/foodiex/)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.6-3178C6?style=for-the-badge&logo=typescript)](https://www.typescriptlang.org/)

## 🚀 Features
- 12 real Pakistani restaurants (KFC, Domino's, Cheezious, Ranchers, Biryani Centre...)
- Restaurant listing with cuisine filters, search, sort
- Full menu with categories, dietary badges, spice levels
- Cart with quantity management, delivery fee calculation
- 3-step checkout with address + payment UI
- Real-time order tracking with animated steps
- Favorites system (Zustand + localStorage)
- Toast notifications
- Mobile-first responsive design

## 🛠 Stack
React 18 · TypeScript · Vite · Tailwind CSS · Framer Motion · GSAP · Zustand · React Router v6
